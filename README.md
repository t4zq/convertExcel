## 🎯 想定質問と回答例

### Q1: なぜC++を使ったのですか？
**A**: データ処理の高速化と、文字列操作の効率化のためです。特に大量のデータを扱う場合、C++のパフォーマンスが重要になります。Emscriptenを使うことで、ブラウザ上でもネイティブアプリに近い速度で動作します。

### Q2: オフラインで使えますか？
**A**:  基本的な変換機能はオフラインで使用できますが、PDFプレビュー機能はTexLive. netのAPIを使用するため、インターネット接続が必要です。将来的には、ローカルでのLaTeXコンパイルにも対応したいと考えています。

### Q3: どのようなデータ形式に対応していますか？
**A**: タブ区切り（TSV）とカンマ区切り（CSV）の両方に対応しています。Excelやスプレッドシートから直接コピー＆ペーストできます。

### Q4: グラフは何種類のデータに対応していますか？
**A**: X軸データ（第1列）と、複数のY軸データ（第2列以降）に対応しています。複数のデータ系列を同時にプロットできます。

### Q5: 他の出力形式への対応予定はありますか？
**A**: 現在、Markdownテーブル形式やJSON形式への対応を検討しています。また、Excelファイルの直接アップロードにも対応予定です。
